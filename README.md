Objective
The goal of this project is to analyze delivery time based on factors like distance, vehicle type, traffic, and weather conditions using Python libraries.
🛠️ Tools & Technologies
Python
Pandas
Matplotlib
Seaborn
📁 Dataset
We used a small dataset of 20 delivery records with the following columns:
Distance (km)
Delivery Time (minutes)
Vehicle Type
Traffic Level
Weather
Key Insights
Delivery time increases as distance increases
Bikes are generally faster than cars and scooters
High traffic causes maximum delay
Weather conditions like rain and fog also increase delivery time
